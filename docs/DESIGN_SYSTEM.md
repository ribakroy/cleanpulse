# CleanPulse Design System

## Brand Direction

CleanPulse משתמש בשפת צבע כחולה שמבטאת:

- מים
- ניקיון
- זרימה
- אמינות
- סביבה בהירה ורגועה

הצבע הירוק הישן אינו צבע מותג ראשי יותר.

## Core Tokens

- `brand`: `#1E88E5`
- `brandWater`: `#38BDF8`
- `brandDeep`: `#0F2742`
- `brandSoft`: `#F4FAFF`
- `background`: `#F4FAFF`
- `border`: `#D8ECFA`
- `muted`: `#64748B`

## Status Colors

- `statusOpen`: כחול בהיר
- `statusAcknowledged`: כחול מים רך
- `statusInProgress`: צהוב עדין
- `statusResolved`: `#6FD29D`
- `statusDismissed`: אפור־כחול רך

## Green Usage Rule

- ירוק מותר רק כצבע משני ל־success.
- אין להשתמש בירוק כצבע brand ראשי.
- כפתורי CTA, active nav, links ו־focus states נשענים על כחול בלבד.

## Shape And Depth

- radius רחב ונעים למגע.
- צללים רכים ועדינים.
- משטחים בהירים עם תחושת SaaS נקייה.

## RTL Rules

- כל המסכים הם `lang="he"` ו־`dir="rtl"`.
- יישור טקסט כברירת מחדל הוא לימין.
- ניווט, טפסים ו־admin shell צריכים להישאר עקביים גם במובייל.

## Mobile And Tablet

- כפתורי kiosk גדולים מאוד עם tap targets נדיבים.
- דשבורד admin נשבר לעמודה אחת במובייל בלי לאבד היררכיה.
- כפתורי פעולה נשארים ברורים גם בטאבלט עומד.
